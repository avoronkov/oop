### Пожелания

- Вместо 1 и 0 для булевых констант лучше использовать `true` и `false`.

- Вместо
```C++
   if (*this == B)
    {   
        return 0;
    }   
    else
    {   
        return 1;
    }   
```
можно написать просто `return !(*this == B);`
